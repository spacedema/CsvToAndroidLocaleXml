# CsvToAndroidLocaleXml

1. How to use
   CsvToAndroidLocaleXml.exe fileToConvert.csv
   CsvToAndroidLocaleXml.exe fileToConvert.csv ,
   CsvToAndroidLocaleXml.exe fileToConvert.csv ;
   
   1st parameter - file name
   2nd - separator

2. Csv format
   Id,          En,           Es,         It
   hello_world, Hello World,  Hola Mundo, Ciao mondo
   
   Wil be generated 3 files: En.xml, Es,xml and It.xml
